# 💌 Will You Be My Valentine?

A cute, interactive Valentine's Day proposal web page — built with pure HTML, CSS, and a sprinkle of charm. Deployed live via GitHub Pages.

🌐 **Live Demo:** [tashkirrr.github.io/Will-You-Be-My-Love-](https://tashkirrr.github.io/Will-You-Be-My-Love-)

---

## 💡 About the Project

This is a fun, romantic mini web project designed to ask that special someone one very important question — *"Will you be my Valentine?"* 🥺

The page features:
- 💌 An **envelope image** as a visual intro
- 🐱 An **animated cat GIF** full of heart-eyes energy
- ✅ A **"Yes"** button — the only right answer
- ❌ A **"No"** button — that *doesn't quite cooperate* (classic Valentine's trick!)
- 📅 A sweet **Valentine's date reveal**: *Burgers at Madchef at 6:30pm. Bring your appetite!*

---

## 🛠️ Built With

- **HTML** — Page structure and layout
- **CSS** — Styling and visual design
- **GitHub Pages** — Free hosting and deployment

No frameworks, no dependencies. Just pure love and code. 💕

## 🚀 Getting Started

Want to run this locally or make your own version?

### 1. Clone the Repository

```bash
git clone https://github.com/tashkirrr/Will-You-Be-My-Love-.git
cd Will-You-Be-My-Love-
```

### 2. Open in Browser

Just open `index.html` in any web browser — no server or installation needed!

```bash
# On Mac
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

## ✨ How to Customize It for Your Valentine

You can easily personalize this for your special someone:

| What to change | Where to find it |
|---|---|
| The Valentine's date/plan | `index.html` — the paragraph at the bottom |
| The envelope or button images | Replace `envelope.png`, `yes.png`, `no.png` |
| The animated GIF | Replace `cat_heart.gif` with any GIF you love |
| Page title / heading text | Edit `<title>` and `<h1>` in `index.html` |
| Colors and fonts | Edit the `<style>` section in `index.html` |

## 🌐 Deploying to GitHub Pages

This project is already live via GitHub Pages. Here's how to set it up for your own fork:

1. Go to your forked repo on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://<your-username>.github.io/Will-You-Be-My-Love-/`
